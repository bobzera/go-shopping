## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/weslley-oliveira/go-shopping.git && cd letmeask-nlw
```

Follow the steps below
```bash
# Install the dependencies
$ npm install

# Start the project
$ npm run dev
```
